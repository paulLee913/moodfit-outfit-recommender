# MoodFit 🎨

날씨와 상황, 원하는 분위기를 선택하면 오늘 입기 좋은 코디를 추천해주는 웹서비스입니다.

## 🌟 서비스 소개

MoodFit은 사용자가 매일 아침 "오늘 뭐 입지?"라고 고민할 때, 복잡하게 옷을 하나하나 비교하지 않아도 날씨와 상황에 맞는 옷차림 방향을 쉽게 결정할 수 있도록 도와주는 서비스입니다.

### 핵심 특징
- **3가지 조건으로 간단한 추천**: 날씨, 상황, 분위기만 선택하면 코디 방향이 나옵니다
- **깔끔한 UI**: 복잡하지 않은 직관적인 인터페이스로 빠르게 선택할 수 있습니다
- **자세한 설명**: 왜 그 코디가 추천되는지, 주의할 점이 무엇인지 알 수 있습니다
- **모바일 최적화**: 휴대폰에서도 부담 없이 사용할 수 있습니다

## 🎯 사용 방법

### 1단계: 조건 선택
다음 세 가지 조건을 선택합니다:

**날씨**
- ☀️ 더운 날씨: 반팔, 얇은 소재 등 시원한 옷
- 🍂 쌀쌀한 날씨: 가벼운 니트, 얇은 아우터
- 🌧️ 비 오는 날씨: 방수 외투, 관리하기 쉬운 신발

**상황**
- 📚 대학 수업: 편안하면서 단정한 분위기
- 🎤 발표: 신뢰감 있고 깔끔한 분위기
- 👥 약속/미팅: 적절한 스타일을 보여주는 분위기
- 💼 면접: 가장 정중하고 신뢰감 있는 분위기
- 🚶 산책: 편안함과 활동성을 우선

**분위기**
- ✨ 깔끔함: 미니멀하고 안정적인 색감
- 🛋️ 편안함: 활동성이 좋은 루즈한 조합
- 👕 캐주얼: 자연스럽고 부담 없는 조합
- 👔 단정함: 셔츠, 슬랙스 중심의 조합
- 🎨 스트릿: 오버핏이나 포인트 아이템이 들어간 조합

### 2단계: 추천 받기
세 가지를 모두 선택하면 **"추천 받기"** 버튼을 누릅니다.

### 3단계: 결과 확인
다음 정보가 제공됩니다:
- **👗 코디 방향**: 추천 옷 조합의 설명
- **🛍️ 추천 아이템**: 구체적인 의류 항목들
- **💡 추천 이유**: 선택한 조건에 맞는 설명
- **⚠️ 주의할 점**: 피해야 할 요소나 조심할 사항

## 📋 실제 사용 예시

### 예시 1: 쌀쌀한 날씨, 대학 수업, 깔끔함
```
선택: 🍂 쌀쌀한 날씨 | 📚 대학 수업 | ✨ 깔끔함

결과:
- 코디 방향: 화이트 또는 크림색 긴팔과 검은색 팬츠
- 추천 아이템: 화이트 긴팔, 검은 팬츠, 로퍼, 미니백
- 추천 이유: 쌀쌀한 날씨에 깔끔한 이미지를 유지합니다.
- 주의할 점: 실내 온도에 따라 얇은 겹겹을 준비하세요.
```

### 예시 2: 비 오는 날씨, 발표, 단정함
```
선택: 🌧️ 비 오는 날씨 | 🎤 발표 | 👔 단정함

결과:
- 코디 방향: 정식 정장에 방수 오버코트
- 추천 아이템: 정식 정장, 방수 오버코트, 정장 구두, 정장백
- 추천 이유: 가장 신뢰감 있고 전문적입니다.
- 주의할 점: 오버코트의 색상이 정장과 잘 어울리는지 확인하세요.
```

### 예시 3: 더운 날씨, 산책, 편안함
```
선택: ☀️ 더운 날씨 | 🚶 산책 | 🛋️ 편안함

결과:
- 코디 방향: 루즈핏 반팔과 편한 쇼츠의 가장 편한 조합
- 추천 아이템: 루즈 반팔, 면 쇼츠, 편한 슬리퍼, 캔버스백
- 추천 이유: 최고의 편안함으로 산책을 즐길 수 있습니다.
- 주의할 점: 너무 집에만 있는 느낌이 되지 않도록 주의하세요.
```

## 🚀 실행 방법

### 설치
```bash
# 프로젝트 클론
git clone https://github.com/paulLee913/moodfit-outfit-recommender.git

# 폴더 이동
cd moodfit-outfit-recommender

# 의존성 설치 (npm 모듈 이미 설치됨)
npm install
```

### 실행
```bash
# 개발 서버 실행
npm start

# nodemon으로 자동 재시작 (개발 시)
npm run dev
```

### 접속
```
http://localhost:3000
```

## 📝 API 엔드포인트

### POST /api/recommend
코디 추천을 받는 엔드포인트

**요청:**
```json
{
  "weather": "hot",
  "situation": "class",
  "mood": "clean"
}
```

**응답:**
```json
{
  "weather": "hot",
  "situation": "class",
  "mood": "clean",
  "direction": "밝은 색상의 얇은 반팔 또는 민소매와 린넨 팬츠의 조합",
  "items": ["흰색/베이지 반팔", "린넨 팬츠", "화이트 스니커즈", "미니멀 가방"],
  "reason": "여름 날씨에 통풍이 잘 되면서도 깔끔한 이미지를 유지할 수 있습니다.",
  "caution": "너무 심플하지 않도록 악세사리로 포인트를 줍시다."
}
```

### GET /api/options
사용 가능한 옵션 목록 조회

**응답:**
```json
{
  "weathers": ["hot", "cool", "rainy"],
  "situations": ["class", "presentation", "meeting", "interview", "walk"],
  "moods": ["clean", "comfortable", "casual", "neat", "street"]
}
```

## 🔄 추천 시스템의 원리

### 날씨의 영향
- **더운 날씨**: 반팔, 얇은 소재, 밝은 색상 추천
- **쌀쌀한 날씨**: 니트, 얇은 아우터, 따뜻한 소재 추천
- **비 오는 날씨**: 방수 제품, 어두운 색상, 관리하기 쉬운 소재 추천

### 상황의 영향
- **대학 수업**: 편안하면서 단정한 스타일
- **발표**: 신뢰감 있고 프로페셔널한 스타일
- **약속/미팅**: 적절한 스타일과 개성의 균형
- **면접**: 최고의 신뢰감과 정중함
- **산척**: 편안함과 활동성 우선

### 분위기의 영향
- **깔끔함**: 흰색, 네이비, 그레이, 미니멀한 조합
- **편안함**: 후드, 니트, 와이드 팬츠, 운동화
- **캐주얼**: 자연스럽고 부담 없는 색감과 조합
- **단정함**: 셔츠, 슬랙스, 정장 분위기
- **스트릿**: 오버핏, 포인트 아이템, 대담한 조합

## ⚠️ 현재 버전의 한계

1. **자동 날씨 정보**: 실제 날씨 API를 연동하지 않아 사용자가 직접 날씨를 선택해야 합니다
2. **개인 맞춤화**: 사용자의 체형, 피부색, 취향을 반영하지 않습니다
3. **옷장 연동**: 실제 보유한 옷장 아이템을 데이터베이스에 저장하지 않습니다
4. **실시간 트렌드**: 계절 트렌드나 최신 스타일이 자동으로 반영되지 않습니다
5. **사용자 피드백**: 추천 결과에 대한 만족도를 수집하지 않습니다
6. **로그인**: 사용자 계정과 추천 히스토리가 저장되지 않습니다
